# AI 360 WORK FORCE Landing Page

Landing page statik untuk kelas AI 360 WORK FORCE.

## Deploy

Fail utama:

- `index.html`
- `style.css`
- `script.js`
- `assets/`

Untuk Vercel, import repo ini dan deploy sebagai static site.

## Environment

Jika sambung fungsi Supabase kemudian, set environment variable di Vercel:

- `VITE_SUPABASE_URL`
- `VITE_SUPABASE_ANON_KEY`

Jangan letak service role key dalam frontend, GitHub atau public repo.
