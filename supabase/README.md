# Supabase Setup

Supabase tidak diperlukan untuk landing page statik ini, tetapi fail `schema.sql` disediakan jika mahu track klik pembelian kemudian.

## Cara guna

1. Buka Supabase project.
2. Pergi ke SQL Editor.
3. Paste kandungan `schema.sql`.
4. Run SQL.

Jangan letak service role key dalam frontend, GitHub atau Vercel public environment.
